---
title: "Upload Files"
---

Quá trình đưa file lên hệ thống bao gồm cơ chế Upload vào kho lưu trữ đám mây (Cloud Storage) và đưa xuống Background Workers (Excel Parser Service) để xử lý lượng lớn dữ liệu ngầm mà không làm treo trình duyệt của bạn. Các bước như sau:

**Bước 1: Chọn chức năng Tải file**

- Trên thanh công cụ bên trái (Menu), bạn hãy nhấn vào nút **Create Dashboard** (Tạo trang tính).
- Màn hình sẽ hiện ra bảng chọn nguồn dữ liệu. Bạn hãy nhấn vào nhóm **File Upload**

**Bước 2: Chọn file từ máy tính**

- Bạn có thể bấm thẳng vào khung màn hình để chọn file từ Máy tính.
- Hoặc đơn giản hơn: **Kéo file từ máy tính và thả trực tiếp** vào khung vuông đó.

**Bước 3: Chờ hệ thống tự động xử lý** Ngay khi chọn xong, tiến trình tải lên sẽ bắt đầu. Bạn không cần làm gì thêm, chỉ cần quan sát thanh tiến trình trên màn hình:

- **Từ 0% - 90% (Đang tải lên):** File của bạn đang được sao lưu an toàn lên kho lưu trữ đám mây.
- **Từ 90% - 100% (Đang xử lý):** Hệ thống đang tự động bóc tách từng cột, từng hàng và đưa vào kho dữ liệu báo cáo sẵn sàng cho bạn sử dụng.
- Thời gian tải phụ thuộc vào dung lượng file và tốc độ mạng mạng internet của bạn (thông thường chỉ mất vài giây).

**Bước 4: Hoàn tất**

- Khi thanh tiến trình báo **100% (Hoàn thành)**, một thông báo xanh sẽ hiện ra báo hiệu "Tải file thành công".
- Ngay lập tức, hệ thống sẽ **tự động chuyển trang** sang màn hình có chứa các dữ liệu trong file bạn vừa tải lên. Tại đây, bạn đã có thể bắt đầu kéo thả các cột để vẽ biểu đồ!
- _(Lưu ý nhánh: Nếu file bạn tải lên bị lỗi (ví dụ sai định dạng hoặc bị lỗi cấu trúc), hệ thống sẽ hiện thông báo màu đỏ để báo lỗi. Bạn chỉ cần sửa lại file ở phần A và tải lại)._
